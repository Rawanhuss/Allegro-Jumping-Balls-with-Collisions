# Allegro-Jumping-Balls-with-Collisions
### To Run this project you must installed allegro by write the command:<br />
### $ sudo apt-get install liballegro4.2 liballegro4.2-dev<br />
### --------------------------------------------------------------<br />
### in the Terminal:<br />
### $ cd JumpingBalls<br />
### $cd ptask-master<br />
### $ mkdir build<br />
### $ cd build<br />
### $ cmake<br />
### $ make<br />
### $ cd examples<br />
### $ sudo su<br />
### $ ./ball<br />
